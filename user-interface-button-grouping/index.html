Editor with Button Grouping user interface

Tidy things up 🧹️ with a customizable toolbar

CKEditor’s toolbar lets you mix and match as many (or as few) features as you need. When things get too busy, you can easily group toolbar buttons. Look, for instance, how this editor’s features are divided into neat groups with | separators.

To unclutter the toolbar even more, you can arrange items into dropdowns. Use the Lists button above to reveal a menu with different list types.

Outstanding configurability 👌

But that’s not all! CKEditor also offers toolbar wrapping : automatic or at a specified breakpoint. This works great with toolbars that do not fit the editor’s width. Additional buttons can either be hidden under a ⋮ menu or moved to the next toolbar line.

You can even add your own buttons to the toolbar by creating custom plugins !
